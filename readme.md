# Foodio

A full stack web application where users can order food from a restaurant and book reservations.

## Tech Used

**Client:** React, TailwindCSS

**Server:** Node, Express, Mongoose

**Database:** MongoDB

**Libraries:** react-router, react-toastify, firebase
